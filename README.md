# HR-analytics-dashboard-tableau
Interactive Tableau dashboard analyzing employee attrition, workforce demographics, job satisfaction, and retention trends.
## Project Overview

This Tableau dashboard provides insights into employee attrition patterns, workforce demographics, age distribution, education background, and job satisfaction levels. The dashboard helps HR teams identify factors influencing employee turnover and make data-driven workforce decisions.

## Tools Used

- Tableau Public
- HR Analytics Dataset
- Data Visualization & Dashboard Design

## Key KPIs

| KPI | Value |
|------|------|
| Employee Count | 1,470 |
| Attrition Count | 237 |
| Attrition Rate | 16.12% |
| Active Employees | 1,233 |
| Average Age | 37 |

## Dashboard Features

- Department-wise Attrition Analysis
- Employee Distribution by Age Group
- Job Satisfaction Rating Analysis
- Education Field-wise Attrition
- Gender-based Attrition Analysis
- Attrition Rate Across Different Age Groups
- Interactive Filters for Workforce Analysis

## Business Insights

1. Overall employee attrition rate stands at 16.12%.
2. Research & Development department recorded the highest attrition count.
3. Most employees fall within the 30–36 age group range.
4. Life Sciences and Medical education backgrounds contribute the highest attrition numbers.
5. Attrition is slightly higher among male employees compared to female employees.
6. Employees aged 25–34 represent the largest workforce segment.
7. Job satisfaction levels vary significantly across different job roles.
